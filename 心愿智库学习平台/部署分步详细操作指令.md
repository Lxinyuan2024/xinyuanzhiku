## 第二步：创建仓库（2分钟）

### 操作步骤：
1. 登录GitHub后，**点击页面右上角的 "+" 号**
2. 选择 **"New repository"**
3. **填写仓库信息**：
   ```
   Repository name: xinyuanzhiku
   Description: 心愿智库学习平台 - 土木工程专业问答平台
   Public / Private: 选择 **Public**（一定要选Public）
   勾选：☑️ Add a README file
   ```
4. **点击 "Create repository"** 绿色按钮

### 成功标志：
看到页面中央显示：**"Quick setup"** 字样
页面网址是：`https://github.com/Lxinyuan/xinyuanzhiku`

## 第三步：上传网站文件（5分钟）

### 方法A：网页直接拖拽（最简单）
1. **找到你的网站文件夹**：
   ```

   导航到：C:\Users\user\WorkBuddy\Claw\心愿智库学习平台\
   ```

2. **回到浏览器打开仓库页面**：
   ```
   访问：https://github.com/Lxinyuan/xinyuanzhiku
   ```

3. **拖拽上传**：
   ```
   选中 "心愿智库学习平台" 文件夹
   按住鼠标拖拽到GitHub仓库页面空白处
   看到绿色虚线框后松手

4. **填写提交说明**：
   ```
   在下方Commit changes那里输入：

### 方法B：如果拖拽失败，用文件选择上传
1. 在仓库页面，点击 **"Add file"** → **"Upload files"**
2. 点击 **"choose your files"**
3. 按 **Ctrl+A** 全选文件夹内所有文件
4. 点击 **"Open"**
5. 填写提交说明，点击 **"Commit changes"**

### 上传成功标志：
看到页面刷新，显示所有上传的文件列表（应该有5-10个文件）

## 第四步：开启GitHub Pages（3分钟）

1. **点击仓库页面顶部的 "Settings"**（在右侧）
2. **左侧菜单找到并点击 "Pages"**
3. **配置部署来源**：
   ```
   Source: 选择 Deploy from a branch
   Branch: 选择 main
   Folder: 选择 /(root)
   ```
4. **点击 "Save"**（页面会刷新）

### 等待部署（2-5分钟）
- 刷新页面几次
- 看到出现绿色提示：**"Your site is published at https://Lxinyuan.github.io/xinyuanzhiku"**

## 🎯 如果遇到问题

### 问题1：拖拽上传时没有绿色虚线框
**解决方法**：
1. 先用Ctrl+A全选文件
2. 点鼠标右键，创建ZIP压缩包
3. 拖拽ZIP文件到GitHub页面
4. GitHub会自动解压

### 问题2：GitHub Pages显示404
**解决方法**：
1. 检查仓库配置：Public仓库、main分支
2. 等待5-10分钟再试（GitHub有缓存）
3. 确保上传的文件夹里有 `index.html` 文件

### 问题3：样式加载异常
**解决方法**：
1. 按F12打开开发者工具
2. 查看Console（控制台）错误信息
3. 截图给我看
